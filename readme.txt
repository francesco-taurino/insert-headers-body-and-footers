Allows you to add extra scripts/codes to the header, after the body tag and the footer of an individual post (or any post type) and/or site-wide, without touching your theme files.

== Frequently Asked Questions ==
 
= Where is the settings? =
1. Menu > Settings > Insert Headers, Body and Footers
2. New/Edit Post > Metaboxes > Insert Headers, Body and Footers

= Scripts not showing up =
Ensure your theme is using `wp_head();`, `wp_footer();` `wp_body_open();` in your templates.

== Installation ==
 
1. Upload the `Insert Headers, Body and Footers` plugin to your WordPress site in the `/wp-content/plugins` folder or install via the WordPress admin.
2. Activate it from the WordPress plugin admin screen.
